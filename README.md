# ✨ SwiftUI Animation Examples
A collection of SwiftUI animation techniques — from basic transitions to spring animations and custom timing curves.

---

## 🤔 What this is
This project walks through the most common SwiftUI animation APIs: implicit animations with `.animation()`, explicit animations with `withAnimation`, spring effects, phase animators, and matched geometry effects. Each example is self-contained so you can drop any piece into your own project. A go-to reference for making your SwiftUI apps feel alive.

## ✅ Why you'd use it
- **Animation variety** — Covers spring, easing, repeat, delay, and custom timing in one place
- **Matched geometry** — Shows how to use `matchedGeometryEffect` for shared-element transitions
- **Copy-paste ready** — Each animation is isolated in its own view so it's easy to extract

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/SwiftUIAnimationExamples.git
cd SwiftUIAnimationExamples
```

### 2. Open in Xcode
Double-click `AnimationExamples.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.

## 🛠️ Notes
- All animations run fine in the Simulator.
- Some newer animation APIs (PhaseAnimator, KeyframeAnimator) require iOS 17+.

## 📦 Requirements
- Xcode 15+
- iOS 16+ (iOS 17+ for newer animation APIs)
